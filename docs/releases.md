# Releases

Public Sprinto releases are published at:

- `https://github.com/AlienPixl/Sprinto/releases`

Release names use the format:

- `Sprinto vX.Y.Z`

Each release includes:

- release notes
- a Linux `amd64` Docker image tarball
- a release-ready Docker Compose file

For update checks, use:

- `UPDATE_REPOSITORY=AlienPixl/Sprinto`
