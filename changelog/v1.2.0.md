## Highlights

- Jira Worklog can now expand reports through selected linked Jira issue types and present the result in a much more report-oriented flow.
- Jira Worklog charts and grouped tables now work together more directly, with stronger theme polish for both Sprinto Classic and Sprinto Glow.

## Changes

- Jira Worklog: added optional linked-issue expansion with a dedicated modal that loads Jira link types, lets users choose exact relationships, and keeps linked-derived rows visibly marked in the final report.
- Jira Worklog: grouping was refined around explicit `Primary group` and `Secondary group` combinations across `Epic`, `Issue`, and `User`, while the old `Group` grouping path was removed.
- Jira Worklog: Epic-aware reporting now preserves parent Epic metadata and links more reliably, including broader-scope reports that are not filtered down to one explicit Epic.
- Jira Worklog: report visuals were reorganized so charts sit above the table, large result tables stay collapsed behind `Show worklog table`, and the legend remains scrollable instead of stretching the whole page.
- Jira Worklog: chart options were simplified to `Donut` and `Treemap`, and chart selection can now highlight matching table rows for faster reading.
- Jira Worklog: the selected chart summary now shows the Jira key, issue or Epic title when available, and a direct Jira link in a new tab.
- Jira Worklog: linked-issue modal layout, chart legend readability, Glow theme contrast, and footer separator visibility were polished so both themes stay readable on desktop and phone-sized layouts.
