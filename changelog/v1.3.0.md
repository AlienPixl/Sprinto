## New Features

- Jira Worklog exports now open from one `Export` menu and support `CSV`, `Excel`, and `PDF`, all generated from the same grouped report table plus the active filter summary.
- Jira Worklog report visuals are now simplified around one donut chart flow, with the toolbar compacted further and `Epic - Include children` moved into the main filter row.
- `Sprinto Glow` is now the canonical theme name across the UI and stored profile preferences, while older saved `timero` values are migrated automatically during load.

## Bug Fixes

- Fixed multiple Jira Worklog mobile layout issues, including legend overflow treatment, issue-picker readability, and linked-issue modal spacing/alignment.
- Fixed Jira Worklog readability problems in `Sprinto Glow`, especially table headers, Jira issue links, and source-column text in grouped report rows.
- Fixed export inconsistencies by removing obsolete chart metadata from export payloads, aligning Excel columns with the grouped table structure, and keeping all export formats focused on the same clean tabular report.
