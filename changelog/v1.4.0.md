
## New features

- Add clickable Jira link for active issue

  - The active issue banner now shows a small link icon which opens the associated Jira issue in a new tab.
  - The issue title remains visually unchanged; the icon provides the link and has keyboard accessibility.
  - Files: `app/web/src/components/RoomView.tsx`, `app/web/src/styles.css`

## Bug fixes

- Filter participants without `vote` permission

  - Users who do not have the `vote` permission are no longer shown as participant cards during voting.
  - Files: `app/src/store.js`

- Jira import panel layout and button sizing

  - Fixed CSS so the import panel buttons no longer stretch the panel and have consistent sizes.
  - Files: `app/web/src/styles.css`

- Round summary layout (Avg / Median)

  - Adjusted the `Round` UI block to allow wrapping and improved sizing so average/median statistics are readable.
  - Files: `app/web/src/styles.css`
