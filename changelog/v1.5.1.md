## Changes

- README overhaul

  - The repository README is now the single public documentation surface. The separate `docs/` and `wiki/` directories have been removed.
  - New full-width hero image with the Sprinto logo embedded directly in the SVG.
  - Shields.io badges for latest release, CI status, license version, and trademarks.
  - Screenshots and video demos are shown in a two-column layout. Four recorded demos cover the Jira import filter builder, vote history timeline, end-to-end voting flow, and the Jira Worklog report.
  - The environment variables table now lists demo account credentials inline with the `SPRINTO_SEED_DEMO_DATA` row so they are easy to find without switching context.
  - The `nginx` reverse-proxy example has been removed from the Quick Start section; it belonged in separate infrastructure documentation that no longer exists.
