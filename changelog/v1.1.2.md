## Highlights

- Dashboard room creation now respects the configured default deck instead of always preselecting the first deck in the list.
- Room settings now make it clearer that `Default timer` is reserved for a future timed-voting feature, and the Classic deck picker hover is visually cleaner.

## Changes

- Dashboard: the create-room deck picker now defaults to the admin-configured `Default deck`, with fallback to the built-in default deck marker and then the first available deck.
- Admin: `Default timer (seconds)` in `Room Settings` is now read-only and labeled as reserved for a future timed voting mode so it no longer suggests active behavior that does not exist yet.
- Dashboard UI: in `Sprinto Classic`, hovering the closed deck picker now highlights only the border treatment instead of filling the whole control with a stronger gold hover background.
