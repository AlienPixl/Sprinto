## Highlights

- Mobile Safari layouts are now stable across topbar navigation, room queue management, Microsoft Entra migration settings, and Jira Worklog date filters.

## Changes

- Room/mobile: `Voting rooms` and `Jira Worklog` now collapse into a compact overflow menu beside the avatar on phone-width layouts.
- Room/mobile: the queue panel now grows with its content on phones, queue rows wrap cleanly, and longer issue lists no longer slide under the footer.
- Admin/authentication: the Entra migration `Default enforce-at date/time` field now uses Safari-safe sizing so iPhone Safari no longer pushes the card wider than its container.
- Jira Worklog/mobile: the `From` and `To` date fields now use the same Safari-safe input treatment and stay inside the report card on iPhone Safari.
